SQL

SQL语句

  • Structured Query Language,结构化查询语音
  • 非过程性的语言:一条语言就对应一个返回结果.

SQL分类

  1. DDL:数据定义语言
    • CREATE, DROP, ALTER
    • 操作表
  2. DML:数据操作语言
    • INSERT, DELETE, UPDATE
    • 操作数据
  3. DCL:数据控制语言
    • GRANT, REVOKE
  4. DQL:数据查询语言
    • SELECT
