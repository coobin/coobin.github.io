CSS属性

Sun, Apr 5, 2020

CSS属性

  • position
    • absolute
      • 脱离原来位置进行定位
      • 相对于最近的有定位的父级进行定位，如果没有，那么相对于文档定位
    • relative
      • 保留原来位置进行定位
      • 相对于自己原来位置进行定位
    • fixed
      • 实现固定定位

定位参照一般设置为relative

广告居中

div{
	position: fixed;
	left:50%;
	top:50%;
	width:100px;
	height:100px;
	background-color:red; 
	margin-left:-50px;
	margin-top:-50px;
}
