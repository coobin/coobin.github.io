习题

浮点运算，比如实现求圆的面积和周长。

expr 只能用于整数计算，可以使用 bc 或者 awk 进行浮点数运算。


#!/bin/bash

raduis=2.4

pi=3.14159

girth=$(echo "scale=4; 3.14 * 2 * $raduis" | bc)

area=$(echo "scale=4; 3.14 * $raduis * $raduis" | bc)

echo "girth=$girth"

echo "area=$area"

以上代码如果想在环境中运行，需要先安装 bc。

$ sudo apt-get update
$ sudo apt-get install bc

题目一

写一个脚本

(1) 提示用户输入一个字符串；

(2) 判断：

如果输入的是quit，则退出脚本；

否则，则显示其输入的字符串内容；

#!/bin/bash

read -t 10 -p "Please enter string: " inputStr
case $inputStr in
quit)
  exit 1
  ;;
*)
  echo "$inputStr"
  ;;
esac

题目三

三角输出

编写bash脚本输出如图的三角

#!/bin/bash
for((i=1;i<=5;i++))
do

  spaceNum=$((5-$i))
  num=$((2*$i-1))

  for ((j=1; j<=$spaceNum; j++))
  do
    echo -n ' '
  done

  for ((k=1; k<=$num; k++))
  do
    echo -n '*'
  done

  for ((l=1; j<=$spaceNum; l++))
  do
    echo -n ' '
  done

  echo ''

done

b=4

猜数字游戏：

首先让系统随机生成一个数字，给这个数字一个范围，让用户猜数字，对输入作出判断，并且给出提示。

请用while语句实现。

#!/bin/bash

function randNum(){
  while :
  do
    read aNum
    if test $aNum -eq $1
    then
      echo "right"
      break 1
    else
      if [ $aNum -gt $1 ]
        then
           echo "The answer is smaller than yours."
         else
           echo "The answer is bigger than yours."
         fi
      fi
  done
}

randNum $(($RANDOM%100+1))
