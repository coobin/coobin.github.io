Git Version Control

Git Syntax

git [command] [--flags] [arguments]

Install Git

  • Windows：GitForWindows
  • Mac：git-osx-installer or brew install git
  • Linux：sudo apt-get update sudo apt-get install git

Setting

Setting user name and email

git config [--local|--global|--system] <key> [<value>]

  • git config --global user.name "YourName"
  • git config --global user.email "YourEmail@xxx.com"

Reading config

git config <key>

  • git user name
  • git user email

Setting gits default editor

  • git config --golbal core.editor nano ## nano is a editor

Git Locations

Creating git location

git init - initialize(create) a repository

Commit to a Local Repository

  • git status View the status of files in the working tree and staging area
  • git add <file or directory> Adds untracked or modified files to the staging area
    • git add . Add all untracked or modified files using git add .
  • git commit Add staged content to the local repository as a commit
    • Prevously committed files are also included
    • Creates a snapshot of the entire project
  • git log View the commit history
    • Limiting the size of log
      • git log --oneline Condensed version of log
      • git log -#Limit the log to the most recent # commits
  • git rm 删除提交的文件并将其重新放到缓存区
